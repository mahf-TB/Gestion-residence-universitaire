# webApp-Cite-universitaire
```
cd backend
php artisan install
```

```
cd frontend
npm install
```


## Setup server PHP
```
php artisan serve
```

### Websocket setup server
```
php artisan websocket:serve 
```

### setup server frontend VueJS
```
npm run serve 
```
